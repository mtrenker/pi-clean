# pi-clean

Custom [pi](https://github.com/mariozechner/pi) package collection.

## Install

```bash
pi install git:git@github.com:mtrenker/pi-clean.git
```

## Structure

| Directory      | Contents                  |
|---------------|---------------------------|
| `extensions/` | TypeScript/JS extensions  |
| `skills/`     | Skill markdown files      |
| `prompts/`    | Prompt templates          |
| `themes/`     | Theme JSON files          |
